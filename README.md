## 🌟 Key Features

### Customer Features
- ✅ Multi-language support (English, French, Kiswahili)
- ✅ Product catalog with categories
- ✅ Multi-select shopping cart
- ✅ Wishlist functionality
- ✅ User accounts with profiles
- ✅ Payment integration
- ✅ Order tracking with delivery status
- ✅ Product reviews & feedback
- ✅ Account deletion (14-day recovery)

### Admin Dashboard
- ✅ Real-time purchase notifications
- ✅ Guest admin management (limited access)
- ✅ Product management (price, pictures, details)
- ✅ Sales analytics by category
- ✅ Website settings control
- ✅ Activity monitoring

### Payment & Orders
- ✅ Multiple payment methods
- ✅ Order confirmation & tracking
- ✅ Delivery status updates
- ✅ Payment history

## 🏗️ Tech Stack

### Frontend
- **Framework:** Next.js (React)
- **UI:** Tailwind CSS
- **State Management:** Redux
- **Multi-language:** i18next
- **HTTP Client:** Axios

### Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** PostgreSQL
- **Authentication:** JWT
- **Payment:** Stripe API
- **Real-time:** Socket.io

### Deployment
- **Frontend:** Vercel (Free tier)
- **Backend:** Railway/Render (Free tier)
